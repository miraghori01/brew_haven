# Brew Haven ☕

Brew Haven is a simple coffee shop website project built using **PHP**, **MySQL**, **HTML**, and **CSS**.
This project includes user registration, login functionality, and a basic interface for a coffee shop website.

## 📌 Features

* User Registration
* User Login System
* Database Connectivity using PHP & MySQL
* Simple and clean user interface
* CRUD operations (Insert, Select, Delete)

## 🛠 Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* XAMPP Server

## 📂 Project Structure

```
Brew_haven/
│
├── index.php
├── login.php
├── register.php
├── insert.php
├── select.php
├── delete.php
├── config.php
└── README.md
```

## ⚙️ Installation

1. Install **XAMPP**.
2. Copy the project folder into `htdocs`.
3. Start **Apache** and **MySQL** in XAMPP.
4. Create a database in phpMyAdmin.

Example:

```
Database Name: mydb
Table Name: users
```

5. Import the database or create table manually.
6. Open browser and run:

```
http://localhost/Brew_haven
```

## 🗄 Database Example

```
Table: users

id (INT, AUTO_INCREMENT, PRIMARY KEY)
name (VARCHAR)
email (VARCHAR)
password (VARCHAR)
```
